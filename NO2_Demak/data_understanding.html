Mengambil data dan menyimpan ke dalam csv#

import openeo
import pandas as pd
import matplotlib.pyplot as plt
import os

# 1Ô∏è‚É£ Koneksi ke Copernicus Data Space
connection = openeo.connect("openeo.dataspace.copernicus.eu").authenticate_oidc()

# 2Ô∏è‚É£ AOI (Area of Interest): DEMAK dan sekitarnya
aoi = {
    "type": "Polygon",
    "coordinates": [
        [
            [110.54853296172155, -6.826224927207619],
            [110.54853296172155, -6.962577241195746],
            [110.71997282462269, -6.962577241195746],
            [110.71997282462269, -6.826224927207619],
            [110.54853296172155, -6.826224927207619],
        ]
    ],
}

# 3Ô∏è‚É£ Ambil data Sentinel-5P (band "NO2") periode selama COVID
s5p = connection.load_collection(
    "SENTINEL_5P_L2",
    spatial_extent={
        "west": 110.54853296172155,
        "south": -6.962577241195746,
        "east": 110.71997282462269,
        "north": -6.826224927207619,
    },
    temporal_extent=["2020-06-01", "2021-06-30"],
    bands=["NO2"],
)

# 4Ô∏è‚É£ Mask nilai negatif (data invalid)
def mask_invalid(x):
    return x < 0

s5p_masked = s5p.mask(s5p.apply(mask_invalid))

# 5Ô∏è‚É£ Agregasi temporal harian
daily_mean = s5p_masked.aggregate_temporal_period(period="day", reducer="mean")

# 6Ô∏è‚É£ Agregasi spasial (rata-rata dalam AOI Demak)
daily_mean_aoi = daily_mean.aggregate_spatial(geometries=aoi, reducer="mean")

# 7Ô∏è‚É£ Jalankan batch job dan hasilkan file CSV
job = daily_mean_aoi.execute_batch(out_format="CSV")

# 8Ô∏è‚É£ Unduh hasil job
results = job.get_results()
results.download_files("no2_results")

# 9Ô∏è‚É£ Baca file CSV hasil
for f in os.listdir("no2_results"):
    if f.endswith(".csv"):
        df = pd.read_csv(os.path.join("no2_results", f))
        print("‚úÖ File ditemukan:", f)
        break

# üîü Pastikan kolom tanggal benar
df["date"] = pd.to_datetime(df["date"])

# 11Ô∏è‚É£ Buat kolom bulan (YYYY-MM)
df["month"] = df["date"].dt.to_period("M")

# 12Ô∏è‚É£ Hitung rata-rata NO‚ÇÇ per bulan
df_monthly = df.groupby("month", as_index=False)["NO2"].mean()

# 13Ô∏è‚É£ Visualisasi hasil
plt.figure(figsize=(10, 5))
plt.plot(df_monthly["month"].astype(str), df_monthly["NO2"], marker="o", color="blue", label="NO‚ÇÇ Rata-rata per Bulan")
plt.title("Rata-rata Bulanan NO‚ÇÇ (Sentinel-5P) di Demak dan sekitarnya")
plt.xlabel("Bulan")
plt.ylabel("Konsentrasi NO‚ÇÇ")
plt.xticks(rotation=45)
plt.grid(True)
plt.legend()
plt.tight_layout()
plt.show()

  1. Koneksi ke Copernicus Data Space

Kode ini menggunakan library openeo untuk terhubung ke platform Copernicus Data Space, yaitu penyedia data satelit dari program Sentinel. Proses authenticate_oidc() digunakan untuk otentikasi agar pengguna bisa mengakses dan mengunduh data.

  1. Menentukan AOI (Area of Interest)

AOI didefinisikan sebagai poligon yang menggambarkan batas geografis wilayah Demak dan sekitarnya menggunakan koordinat lintang dan bujur. Poligon ini menentukan area di mana data NO‚ÇÇ akan diambil dari citra satelit.

  1. Mengambil Data Sentinel-5P

Mengambil data satelit dari produk SENTINEL_5P_L2 dengan fokus pada band NO2, yaitu nitrogen dioksida ‚Äî indikator utama kualitas udara. Data diambil untuk rentang waktu Juni 2020 hingga Juni 2021, yang mencakup masa pandemi COVID-19.

  1. Masking Nilai Negatif (Data Invalid)

Nilai NO‚ÇÇ negatif tidak valid secara fisik (karena konsentrasi gas tidak mungkin negatif). Oleh karena itu, fungsi mask_invalid() digunakan untuk memfilter dan menyingkirkan nilai-nilai tersebut.

  1. Agregasi Temporal Harian

Data satelit biasanya tersedia per scene (tangkapan citra per waktu tertentu). Proses aggregate_temporal_period(period=‚Äùday‚Äù, reducer=‚Äùmean‚Äù) menghitung rata-rata nilai NO‚ÇÇ setiap hari agar menghasilkan data dengan resolusi temporal harian.

  1. Agregasi Spasial

Setelah data diolah secara temporal, langkah berikutnya menghitung rata-rata nilai NO‚ÇÇ dalam seluruh area poligon Demak. Ini dilakukan dengan fungsi aggregate_spatial(), menghasilkan satu nilai rata-rata per hari untuk seluruh wilayah tersebut.

  1. Eksekusi Batch Job & Ekspor ke CSV

Data hasil agregasi diolah di server Copernicus, kemudian hasilnya diekspor menjadi file CSV dengan perintah execute_batch(out_format=‚ÄùCSV‚Äù).

  1. Mengunduh Hasil ke Folder Lokal

Setelah job selesai, hasil unduhan CSV disimpan otomatis ke folder lokal bernama no2_results.

  1. Membaca File CSV Hasil

Program mencari file hasil CSV dalam folder no2_results, membaca isinya menggunakan pandas, dan memuatnya ke dalam variabel df.

  1. Konversi Kolom Tanggal

Kolom tanggal diubah ke format datetime agar dapat digunakan untuk analisis waktu, seperti membuat agregasi bulanan.

  1. Menambahkan Kolom Bulan (YYYY-MM)

Dari tanggal harian, dibuat kolom baru berisi format periode bulanan untuk mempermudah perhitungan rata-rata tiap bulan.

  1. Agregasi Bulanan

Mengelompokkan data berdasarkan bulan (groupby(‚Äúmonth‚Äù)) dan menghitung rata-rata NO‚ÇÇ untuk setiap bulan.

  1. Visualisasi Data

Grafik garis menampilkan tren perubahan rata-rata konsentrasi NO‚ÇÇ per bulan di wilayah Demak. Sumbu X menunjukkan waktu (bulan), dan sumbu Y menunjukkan konsentrasi NO‚ÇÇ. Visualisasi ini membantu memahami pola kualitas udara dalam periode yang diamati.
